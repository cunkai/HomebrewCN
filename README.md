 **---Brew介绍** 

macOS 和 Linux 缺失软件包的管理器

 **---Homebrew 能干什么?** 

使用 Homebrew 安装 Mac（或Linux）没有预装但你需要的东西。

 **--Homebrew自身如何使用** 

知道软件包具体名称，直接 `brew install 软件包名`
只知道一小部分名称，用 `brew search 小部分名称` 查询即可
例如`brew search chrome`就会把带chrome的软件包全部列出
